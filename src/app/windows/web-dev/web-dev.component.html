Projects

Property Scraper

Midway through 2022 I need a place to stay near university. I wanted to be alerted of new properties on the market for renting. I then build an app that would alert me of such properties.

Architecture/Stack: The project is built with to be cross platform with the ionic framework. A self wrote python script is used to collect properties from the web and store them. Another python script is used to send notifications to notification subscribers (me). Google's Firebase Cloud Messaging service is used to deliver payloads to subscribed devices.

Waitless Digital Menu Platform

This is a website where users can view a Menu of a Restaurant. All Restaurants have a specific style and theme of menu and this has to be replicated in the digital menu while not being a static png/svg

Architecture/Stack: The Project uses Nginx on a hosted server to serve the html content. Menu and Restaurant data is stored on a MySQL database. Images are Stored on Google Cloud Buckets

Portfolio Website

This is the website that is currently being viewed. I hope it is up to par

Architecture/Stack: This project makes heavy use of Amazon Web Services. First the source code is compiled to the build files, and these build files are sent to an s3 bucket. The DNS is configured on Route 53 to fetch content from the Cloudfront CDN, which gets its content from the s3 bucket. Using git hooks, I have automated the process of invalidating the cache once new content is uploaded to the s3 bucket.
